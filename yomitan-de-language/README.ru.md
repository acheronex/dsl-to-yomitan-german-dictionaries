# Поддержка немецкого языка для Yomitan

Эта папка содержит файлы для включения деинфлексии немецких слов в Yomitan.

## Файлы

- `german-transforms.js` — правила деинфлексии немецких слов для Yomitan

## Что это?

По умолчанию Yomitan ищет точные совпадения в словаре. Для немецкого языка это означает, что если у вас есть статья для слова "laufen", но вы ищете "lief" или "gelaufen", Yomitan не найдёт её.

Этот файл добавляет правила деинфлексии немецких слов, которые преобразуют изменённые формы слов обратно в словарные:
- **Глаголы**: "lief" → "laufen", "gelaufen" → "laufen", "zu lesen" → "lesen"
- **Существительные**: "Männer" → "Mann", "Tische" → "Tisch"
- **Прилагательные**: "schnellste" → "schnell", "kältesten" → "kalt"
- **Умлауты**: "häufig" → "haeufig", "Bäume" → "Baume"

## Установка

1. Найдите папку вашего расширения Yomitan:
   - **Chrome**: `chrome-extension_[ID]/js/language/de/`
   - **Firefox**: `moz-extension_[ID]/js/language/de/`
   
   Чтобы найти ID, откройте настройки Yomitan в браузере и посмотрите на URL, или найдите `german-transforms.js` в папке расширений.

2. Сделайте резервную копию оригинального файла `german-transforms.js` (если он существует)

3. Скопируйте наш `german-transforms.js` в папку `js/language/de/`

4. Перезагрузите Yomitan

## Текущие ограничения

Это решение, созданное сообществом, и может не охватывать все формы немецких слов. Известные ограничения:

- Не все неправильные формы глаголов охвачены
- Некоторые частные случаи могут не работать
- Обработка умлаутов базовая (ä→a, ö→o, ü→u)

Не стесняйтесь улучшать этот файл и вносить свой вклад!
